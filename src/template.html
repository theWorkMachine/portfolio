Smart-Face-Detection

  • A smart face detection web app that uses an image link to load an image from anywhere on the internet and detects a face in it with the help of machine learning using an API from Clarifai.

  • Smart-Face app lets you sign up with secure password using bcrypt and keeps a count for the times you scan an image using your profile.

  • The front-end of app is built on html5, css3, advanced JavaScript, React.js and Redux.js.

  • Node.js and Express.js were used as server languages along with restful APIs and the database is built using PostgreSQL.

  • Smart-Face detection web app is deployed on Heroku to be used from anywhere
